Direction on How to Use Interface

1) Press the Button that Say Turn ON

2) Select a preset Time and Temperature or select custom Time and Temperature


If Selected Custom

3) Select A Mode. (If one of the preset Mode are selected interface will skip the set of Temperature)

4) If Custom Mode is selected user can select where heat source will be coming from

5) User Input their desire temperature

6) User Input their desire time


If Selected Preset

3) User can go back and change the time and temperature in preset if need


User is shown a confirm page. User can choose to preheat or start


PREHEAT

User input Desire temperature

Toaster Oven is preHeated but if User doesnt not press any button in 10 sec(should be 5 min but is change in app for testing) the toaster oven will turn off


START

Show user the temp inside and The Remaining time until complete

Once Complete user is require to press Done
