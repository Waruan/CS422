User Profile

We used icons for accessing any setting on application on the mirror because icons have become universal, anyone can tell the

weather icon by looking at the sun or the music icon by looking at the music note. We made preferred language a setting that

can later be changed. There is no gender difference for mirrors. We expect the user to be at a reasonable age as anyone

who can use a smartphone can use the mirror with ease.

novice / first time users

First time users will have an easy time understanding using our mascot as a way to ask for help. The bird will be there only during the first time you use the mirror. Also buttons will be easily

recognizable. Consistency with buttons and layout. Buttons are in arm length for easy access.

knowledgeable intermittent users

Just like a phone icons are easily recognizable. Layout is easy to understand and navigate.

Simple actions to make it easier for the user to perform tasks efficiently.

expert frequent users

Simple to navigate on a day to day basis. Few clicks to get from one app to another on the mirror.

Very user friendly in terms of layout and color.

diversity of tasks

Settings similar to something you would see on a phone. Basic tasks such as calender, music and weather.

We allow toggling icons rather than dragging them.
