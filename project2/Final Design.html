Extra info

All the button for function should all be within arm reach

For all social media function the user will be ask to input user and password the first time they use it

Start/User Selection


This is the first screen you will see. In order to access applications, the user must either select an existing account and input their pin, select the guest account, or create a new user.


Create new user


When pressed the "New User" button, the user is prompted to input their name. There will be a keyboard for them to use and the name will be displayed at the middle of the screen like the image.


Input pin for new user


When creating a new user, after you input your name the user will then be prompted to input their pin to be used to access accounts later on.


Guest User


Selecting the guest user will bring you to a default front view. The applications you see at the bottom are the only ones they can access.


Exisitng User Login


When selecting an existing user, it will prompt the user to enter their specific pin to that account. There will be a number pad displayed in the middle of the screen where the user can then input their pin that they created during the initial phase of creating a new user.

Adding New Widget


Pressing on the 9gag icon brings up the mobile version of the 9gag application. The application is read only, meaning users can upvote and everything but they are unable to past any comments or memes online.


After adding new widgets


This shows what happens after you select an app to add onto the front page. Notice the "News" application is now transparent with a check mark. The icon itself is now displayed on the front page.


Facebook


Pressing the facebook application will bring up a mobile version of facebook. The only difference is that the application will only allow a "read only" feature. As in the user can't post of comment anything. The user can only look through their facebook news feed and that is all.


Moving Popup


Moving an application by pressing and holding will display a gray area the user will see. This is the area where you can move widgets around. This will provide the user with a limited space to move things around and to prevent things from going off screen.


After moving popup


This is how the screen will look after you move the app in a different position. Once you let go, the gray area will disappear and the mirror will return back to normal.


Hide Everything Button


The hige button will hide all app icons from the bottom display, showing only the mirror and the time/date/weather on the top left. To bring back all icons, press the hide button again.


Calender


When you press the calender icon, the display will bring up a full calender where you can select a specific date. On the right side of the widget, it will display the date as well as any notes you have for that day. A keyboard will be at the bottom of the widget to allow the user to edit anything on that date.


Music


Pressing on the music icon will bring up a music widget. The user logs into Spotify and the playlist will automatically be loaded onto the mirror from your account. The music widget will allow the user to select songs and/or play songs


Small Music Interface


When music is still playing and the user taps out of does something else, the music player will shrink and be placed on the right side for ease of access while the music continues to play. To open the music widget, press the music icon again.


Timer


Start/Selecting the timer icon at the bottom will bring up the timer widget. Here the user can set up a timer for various activities. When pressing off the screen, the timer will shrink.


Small Timer Interface


The timer is minimized and put to the right of the screen. The number will indicate how much time is left. TO access the timer again, press the timer icon at the bottom.


Twitter


Pressing the Twitter icon will display your twitter new feed.


9gag


Pressing on the 9gag icon brings up the mobile version of the 9gag application. The application is read only, meaning users can upvote and everything but they are unable to past any comments or memes online.


New feed


When you select the news icon at the bottom, this will display the news application. At the bottom, there will be different sections for the user to pick. Each selection will display several articles where the user can the select.


New Feed extended


When you select a News Article, the display will extend, showing the user the detailed version of the article


health


When pressed the health icon, a display will show up in the middle of the screen. At the bottom area, there will be selections for Sleep, Steps, and weight. Pressing each tab will display information regarding to each cataory.Data will be gathering from device like fitbit


Weather


Selecting the weather icon at the bottom will bring up the weather application. Here we can see a detailed forcast. The user will be allowed to select weekly, hourly, today, and a map option. The map wil give the user a satalite image of what the weather looks like.


Setting


This is the settings menu. The user will have the ability to modify the mirror based on what is displayed on this menu.


Bluetooth


The bluetooth setting will display a box that will contain all available Bluetooth devices in the current area. In order to connect with the device, simply select the device you wish and it will automatically connect.


Time


Selecting the time option, the user will then have the ability to set their time. The UI design is based off the android system so that most people will be familar with how the set up works. The user is to first selct the hourly time by pressing on a number on the clock, then minutes using the same method.


Personalization


This will allow the user to make changes to their account. Meaning, they can have two options. The ability to change their account name and their pin. Selecting either or will bring you to a different section to continue that action


Change name


This selection will allow the user to change their name during the initial user selection. The display will prompt for the user to enter a new name and a keyboard will be provided


Change pin


The user will be prompted to enter their existing pin. After that passes, the user will be prompted to enter a new pin they wish to change it do.


Contrast


This setting is to set the contrast level of the entire UI.


Icon Resize


This setting will allow the user to resize the icon/text to the desired size.


Language


The user will be allowed to select different languages they wish. The setting will display all languages the software supports and the user is to simply press on the desired language.


New Filter


This section is to modify the information displayed in the News Article application. Here the user can select their news source, and any filters they wish. THe picture shows a filter of Business and gaming, but can be changed dpending on the need of the user.


Location


The user will be able to select their location


Transparency


This setting will allow the user to set the level of transperency. Moving the bar left and right will either make the screen more or less transparent. The user will not achieve complete transparency due to the issue of not being able to see anything.


Wifi


This is the Wireless internet section. In order to turn on the WiFI settings, press the button at the top until it says "ON". The user will have a list at the bottom that will show all existing wireless internet the mirror has found. Pressing on any of the networks will prompt the user to enter the password. THe user will be provided a keyboard to enter their password.
